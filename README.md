# Installation
  
### Prerequisites
- Node.js 18+
- npm or yarn
- WhatsApp account

### Quick Start
```bash
# Clone repository
git clone https://github.com/MidknightMantra/Mantra.git
cd Mantra

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your settings

# Start bot
npm start
```

### Docker Deployment
```bash
# Build and run
docker-compose up -d

# View logs
docker-compose logs -f

# Stop
docker-compose down
```
